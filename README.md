# Old Ukrainian Dicitonary Stemmer

Скрипт для извлечения основ из староукраинских словарей на основе двухтомника Тимченко (Тимченко, Євген Костянтинович. Матеріали до словника писемної та книжної української мови XV-XVIII ст. Книга 1, книга 2)

## Описание

Использует пары "слово - часть речи" для усечения окончаний и стемминга. На выходе выдает json-файл с лексемами в ключах и списками возможных основ в значении.
